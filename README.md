# Element Discord User Style
A user style for the Element web app that themes it to look closer to Discord

## Install

**Note:** If you use the Styl***ish*** extension, it is recommended to uninstall that and use the open source Stylus extension instead due to the [privacy issues](https://en.wikipedia.org/wiki/Stylish_(software)#Privacy_issues) with Stylish.

1. Install the Stylus extension:
   * [Chrome Web Store](https://chromewebstore.google.com/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne) - Chrome, Opera, Edge, or other Chromium based browsers
   * [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/styl-us/) - Firefox or other Gecko based browsers
3. Open the raw user style from the files of this repo, or [![Install directly with Stylus](https://img.shields.io/badge/Install%20directly%20with-Stylus-00adad.svg)](https://github.com/jessedufrene/element-discord-userstyle/raw/refs/heads/main/element-io-discord-theme.user.css)
4. (recommended) Install [element.io](https://app.element.io) as a web app (Usually by pressing a button at the right of the address bar. [How to on Chrome](https://support.google.com/chrome/answer/9658361))
5. (recommended) Enable notifications in element.io ([How to](https://docs.element.io/latest/element-support/element-webdesktop-client-settings/notification-settings/#desktop-notifications))
